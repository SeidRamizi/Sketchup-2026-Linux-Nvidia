## ⚠️ Prerequisites (MUST INSTALL FIRST)
Before running the script, make sure you have **Wine** and **Winetricks** installed on your system:

### For Ubuntu / Linux Mint / Pop!_OS:
```bash
sudo apt update && sudo apt install wine winetricks -y
```

### For Fedora:
```bash
sudo dnf install wine winetricks -y
```

### For Arch Linux / Manjaro:
```bash
sudo pacman -S wine winetricks --noconfirm
```
